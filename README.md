# Controle de Candidaturas

Sistema web para gerenciamento de candidaturas de emprego.

## Funcionalidades

- Cadastro de vagas
- Cadastro de empresa
- Data e horário de entrevistas
- Controle de status
- Exclusão de registros
- Salvamento automático com LocalStorage

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
